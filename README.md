# Home Made Cake - Natore

This is a premium, one-page website designed for **Home Made Cake**, a bakery based in Nichabazar, Natore.

## Features
- **Modern Design**: Soft pastel pink theme with a professional and elegant "homemade" feel.
- **Mobile First**: Fully responsive layout optimized for smartphones.
- **WhatsApp Integration**: Direct ordering button via WhatsApp.
- **Interactive Gallery**: A grid to showcase your beautiful cake creations.
- **Dynamic Scroll**: Smooth scrolling navigation and fade-in animations.
- **Google Maps**: Embedded location for Nichabazar, Natore.
